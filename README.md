# 📰 Fake News Detection System

A Machine Learning project developed using **Python** and **Scikit-learn** to classify news articles as **REAL** or **FAKE**. The model uses **TF-IDF Vectorization** and the **Passive Aggressive Classifier** for text classification. A simple web interface is also created using **Gradio** for real-time prediction.

---

## 📌 Project Overview

The Fake News Detection System analyzes news articles or headlines and predicts whether they are **REAL** or **FAKE** based on trained machine learning models. The project demonstrates the complete machine learning workflow, including data preprocessing, feature extraction, model training, evaluation, and deployment.
